# Book Tracker

**Автор:** Муратов Денис
**Язык:** Python 3.10

## Описание программы

Book Tracker — простое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Есть фильтрация по жанру и количеству страниц. Все данные сохраняются в файл `books.json`.
## Инструкция по запуску

1. Установите Python 3.10
2. Скачайте или склонируйте репозиторий.
3. Запустите файл `book_tracker.py`:
   ```bash
   python book_tracker.py
   ```
4. Используйте интерфейс для добавления и фильтрации книг.
Примеры использования
1. Добавление новой книги
Введите данные в поля формы:
Название: «Мастер и Маргарита»
Автор: Михаил Булгаков
Жанр: Фантастика, Сатира
Количество страниц: 400
Нажмите кнопку «Добавить книгу».
Книга появится в таблице.
2. Фильтрация по жанру
Введите в поле «Жанр» ключевое слово, например: «фант».
Нажмите «Применить фильтр».
В таблице останутся только книги, в жанре которых есть это слово (например, «Фантастика»).
3. Фильтрация по количеству страниц
Введите в поле «Страниц >» число, например: 300.
Нажмите «Применить фильтр».
В таблице отобразятся только книги, объем которых превышает 300 страниц.
4. Комбинированная фильтрация
Вы можете использовать оба фильтра одновременно:

Введите жанр (например, «детек» для детективов).
Введите минимальное количество страниц (например, 500).
Нажмите «Применить фильтр».
Отобразятся только детективы объемом более 500 страниц.
5. Сохранение и загрузка данных
При закрытии окна программы все добавленные книги автоматически сохраняются в файл books.json.
При следующем запуске приложения список книг загрузится из этого файла автоматически, поэтому данные не потеряются.
Пример внешнего вида интерфейса:
Главное окно: Поля для ввода данных, кнопка добавления и таблица со списком книг.
Окно с фильтром: Поля для ввода жанра и количества страниц, кнопка применения фильтра.
