# Book Tracker

**Автор:** Муратов Денис
**Язык:** Python 3.10

## Описание программы

Book Tracker — простое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Есть фильтрация по жанру и количеству страниц. Все данные сохраняются в файл `books.json`.

## Инструкция по запуску

1. Установите Python 3.x (если не установлен).
2. Скачайте или склонируйте репозиторий.
3. Запустите файл `book_tracker.py`:
   ```bash
   python book_tracker.py
   ```
4. Используйте интерфейс для добавления и фильтрации книг.
