# Expense-Tracker

**Автор:** Муратов Денис  
**Описание:** Простое приложение для учёта личных расходов с фильтрацией по категориям и датам, подсчётом суммы за период и сохранением данных в JSON.
**Язык:** Python 3.11  
**Библиотеки:** Tkinter, tkcalendar

# функционал
- Добавление расходов (сумма, категория, дата).
- Фильтрация по категории.
- Подсчёт суммы за выбранный период.
- Сохранение и загрузка данных из `expenses.json`.
- Проверка корректности ввода.
